# Assets Required

Please place the following image files in the `assets/` folder:

## Logo Images
- `Moh_logo.png` - Main logo for top left of navigation
- `Moh_text.png` - Text logo for center of navigation

## Social Media Icons
- `facebook_logo.png` - Facebook icon for footer
- `instagram_logo.png` - Instagram icon for footer

## About Page Images
- `founder_image.jpg` - Image of the founder for the about page story section
- `team_image.jpg` - Team photo for the about page story section

All images will be automatically integrated into the website once placed in the assets folder.
